# UI Tokens

## Purpose

This file defines the visual foundation of ScribbleBox.

These tokens are the source of truth.

Do not hardcode colors, spacing, typography, radius values, or shadows.

Always use design tokens.

---

# Design Personality

ScribbleBox should feel:

- playful
- warm
- handmade
- personal
- scrapbook-inspired

The UI should feel like a physical notebook rather than a software dashboard.
