# Creative Design Agency — Design System Spec

## Colors
- **Canvas**: `#000000` (deep pitch black)
- **Ink**: `#ffffff` (white)
- **Accent**: `#ff003c` (neon crimson/red)
- **Border**: `#18181b` (zinc gray)

## Typography
- **Headings**: Saturated grotesque display, uppercase, weight `900`
- **Body / Subtexts**: JetBrains Mono for taxonomy eyebrows and description tags

## Motion
- **Interactions**: Tab active state transfers on project hover, completed inside `150ms`.
- **Entrance**: Staggered lists reveals using custom spring curves.
