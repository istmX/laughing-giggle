# Developer Portfolio — Design System Spec

## Colors
- **Canvas**: `#faf8f5` (warm linen/sand)
- **Ink**: `#1c1917` (deep stone)
- **Accent**: `#8b5cf6` (purple)
- **Border**: `#e7e5e4` (soft tan hairline)

## Typography
- **Display Headings**: Fraunces (serif, italic option, weight `300`/`400`)
- **Body Copy**: Inter (sans, weight `320`/`330` for legibility)
- **Measure**: Line length capped at `70ch` to maintain rhythm.

## Layout & Motion
- **Filtering**: Project grid filter transitions with layout animation constraints under `300ms`.
- **Newsletter**: Slide-in transitions on active subscription state confirmation.
