# B2B SaaS Kanban Board — Design System Spec

## Colors
- **Canvas**: `#09090b` (zinc black)
- **Ink**: `#f4f4f5` (zinc off-white)
- **Border**: `#18181b` (dark slate hairline)

## Components & States
- **Kanban Card**: Hover raises card details slightly, active focus adds zinc outlines.
- **Form Controls**: Inputs have strict 48px tap target minimums.
- **Sidebar**: Neutral secondary panel background color slightly lighter than main canvas.

## Motion
- **Flow**: Column transfers completed inside `150ms-250ms` using exponential decels.
