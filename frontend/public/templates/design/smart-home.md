# Mobile Smart Home App — Design System Spec

## Colors
- **Canvas**: `#0c0c0e` (pure mobile dark)
- **Ink**: `#ffffff` (white)
- **Accent Primary**: `#f59e0b` (amber warm yellow)
- **Border**: `#27272a` (zinc border)

## Components
- **Device Cards**: Dynamic backdrop opacity changes based on active states.
- **AC Target Sliders**: Interactive input ranges allow target selection.
- **Bottom Navigation**: Tabs allow screen transitions (Home, Rooms, Settings).
