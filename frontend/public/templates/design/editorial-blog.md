# Minimalist Editorial Blog — Design System Spec

## Colors
- **Canvas**: `#ffffff` (pure white)
- **Ink**: `#18181b` (zinc black)
- **Border**: `#f4f4f5` (hairline soft gray)

## Typography
- **Headings**: Fraunces (serif, italic option, weight `400`)
- **Prose Copy**: Inter (sans, weight `320`/`330`)
- **Line Gaps**: Target line height set at `1.60` for paragraphs.

## Layout Options
- **Font Scale**: Dynamic font size toggles allow users to adjust base paragraph reads from 14px to 24px.
