# ScribbleBox Mobile Project Overview

## Product Name

ScribbleBox Mobile

---

# Vision

ScribbleBox Mobile transforms native iOS and Android screenshots into structured, searchable memories.

People take dozens of mobile screenshots every week.
Most are forgotten.
Months later they have no idea:
- why they saved them
- what they meant
- why they mattered

ScribbleBox Mobile solves this by letting users organize screenshots into visual mobile collections and attach context notes directly on their phones.

---

# Problem Statement

Modern mobile photo rolls become cluttered with screenshot noise.

Examples:
- mobile app design snippets
- chat conversation screenshots
- travel directions & map pins
- shopping receipts & study notes

Users remember saving them, but forget WHY.

---

# Solution

Every mobile screenshot becomes a structured memory:
- Mobile Screenshot + Context Note + Collection Tag + Pinned Marker = **Preserved Memory**

---

# Target Audience

Primary Users:
- mobile power users
- designers & developers on the go
- students & researchers

---

# Core User Journeys

1. **Tab Navigation Journey**: Seamless switching between Timeline, Collections, and Settings tabs.
2. **Screenshot Import & Context Attachment**: Direct media picker import with immediate note attachment.
3. **Memory Inspection Journey**: Tapping a memory card pushes a detailed view stack screen with a native back button gesture.

---

# Primary Screen Specifications

- **Home / Timeline Tab Screen**: Greeting header, pinned memories widget, recent screenshot thumbnails, and calendar timeline grouping.
- **Collections Tab Screen**: Visual grid of color-coded mobile memory collections.
- **Screenshot Details Stack Screen**: High-res image preview, multiline note editor, collection tag selector, pin toggle.
- **Context Note Modal Screen**: Slide-up modal sheet (`rounded-t-3xl`) for quick note entry.
- **Settings Tab Screen**: Local storage usage stats, export backup button, clear data trigger.

---

# Success Metric

The app succeeds when mobile users can answer:
"Why did I save this screenshot?" on their phones without guessing.
