# Zenix Project Overview — Mobile App Blueprint

Core product vision, mobile problem statement, and non-negotiable user journeys for ScribbleBox Mobile.

---

# 1. Product Vision & Problem Statement

### Vision
ScribbleBox Mobile brings screenshot context preservation to iOS and Android devices, allowing users to save and curate memories anywhere.

### Problem Statement
Users take dozens of mobile screenshots every week. Without context, these screenshots fill mobile photo galleries with digital noise. ScribbleBox Mobile captures the thought, note, and memory behind every screenshot.

---

# 2. Key Mobile User Journeys

1. **Tab Navigation Journey**: Seamless switching between Timeline, Collections, and Settings.
2. **Screenshot Import & Context Attachment**: Direct media picker import with immediate note attachment.
3. **Memory Inspection Journey**: Tapping a memory card pushes a detailed view stack with full-screen image and notes.
