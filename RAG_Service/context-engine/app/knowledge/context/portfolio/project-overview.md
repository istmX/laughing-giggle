# Zenix Project Overview — Visual Portfolio & Showcase Blueprint

Executive vision statement, target audience, non-negotiable user journeys, and screen specs for ScribbleBox visual portfolios.

---

# 1. Product Vision & Problem Statement

### Vision
ScribbleBox Showcase presents screenshot collections, memory galleries, and creative engineering work in a high-impact, visual interface.

### Problem Statement
Static portfolio grids lack visual storytelling and context. ScribbleBox Showcase combines high-contrast typography, interactive category filters, and motion reveals to make portfolios engaging.

---

# 2. Key User Journeys

1. **Hero Section Entry Journey**: Sticky glassmorphic navbar, looping marquee ticker, display title, and 70vh cover banner.
2. **Gallery Filter & Modal Journey**: Category filter tabs (`All`, `Web App`, `Design`), stagger card reveals, and slide-over detail modals.
3. **Contact & Social Connect Journey**: Footer social links and email inquiry button.
