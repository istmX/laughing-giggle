# Zenix Project Overview — Full-Stack SaaS Blueprint

Executive vision statement, target audience, problem definition, and non-negotiable user journeys for ScribbleBox SaaS.

---

# 1. Product Vision & Problem Statement

### Vision
ScribbleBox SaaS transforms chaotic screenshot collections into organized, searchable memory workspaces.

### Problem Statement
Modern users save hundreds of screenshots containing app ideas, research snippets, and product designs. Without context, these screenshots become meaningless digital clutter. ScribbleBox SaaS attaches context, notes, and tags to screenshots, making them instantly useful.

---

# 2. Key User Journeys

1. **Authentication & Onboarding**:
   - User signs up/logs in securely.
   - User selects memory storage workspace and sets preferences.

2. **Memory Capture & Context Enrichment**:
   - User imports screenshots.
   - AI assistant extracts metadata, tags, and generates contextual notes.

3. **Workspace Organization & Search**:
   - User categorizes memories into collections.
   - Fast full-text and tag-based search across all saved memories.
