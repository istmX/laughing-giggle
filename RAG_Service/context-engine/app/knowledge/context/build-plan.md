# MVP Definition

MVP includes:

✓ Onboarding

✓ Authentication

✓ Import screenshots

✓ Collections

✓ Notes

✓ Timeline

✓ Settings

Anything else is optional.

---

# Out of Scope

Not MVP:

- OCR
- Cloud Sync
- Sharing
- Collaboration
- Social Features

Ignore these until requested. AI orchestration, conversational prompts, project chat, and context generation are explicitly in-scope.
